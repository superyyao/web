
- Bower常用命令

```shell
$ # 初始化一个Bower的配置文件
$ bower init
$ # 安装一个包
$ bower install bootstrap
$ # GitHub shorthand
$ bower install desandro/masonry
$ # Git endpoint
$ bower install git://github.com/user/package.git
$ # URL
$ bower install http://example.com/script.js
$ # 安装一个包并将其添加到配置文件
$ bower install bootstrap --save
$ # 卸载一个包
$ bower uninstall bootstrap
$ # 更新所有的包
$ bower update
```
